#### 🧠 后端开发任务：单轨文本抽取核心 (API 与 Prompt)
**目标：先不管线上实时追问，这周死磕“大段会议文本”瞬间提炼四维标签的逻辑。**

* **Task 1：开发 `/extract-persona` 核心接口**
    * **开发内容：** 写一个新的接收接口，允许前端传入一段超长文本（模拟会议纪要）。
* **Task 2：Prompt 炼丹（本周重头戏）**
    * **开发内容：** 针对 PM 提供的会议纪要测试集，编写一套强大的 System Prompt。要求大模型阅读千字废话后，准确无误地吐出 PM 规定的四维 JSON 格式，绝不多加废话和 Markdown 符号。
* **Task 3：开发 `/compile-firmware` 雏形接口**
    * **开发内容：** 接收一段 JSON 标签，套用预设的模板字符串，输出一段用来给机器人的“系统指令文本”（简单组装即可，跑通输入输出流）。

📦 **【周五 PM 验收标准】**
不需要看前端界面。后端哥们直接用 Postman 或 Swagger 演示：把 PM 写的 1000 字混乱会议纪要复制进去发起请求，接口在 5-10 秒内，**稳定返回格式完美、提取精准的 4 维度 JSON 标签。**
